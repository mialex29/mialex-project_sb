Seni dalam Kode

Belajar bagaimana teknologi memungkinkan kita menciptakan seni digital yang menakjubkan. Dari algoritma fraktal hingga generative art, coding membuka pintu kreativitas tanpa batas.

Contoh Kode Seni (HTML5 Canvas):


function setup() {
    createCanvas(800, 600);
    background(240);
    noStroke();
    
    for (let i = 0; i < 100; i++) {
        fill(random(255), random(255), random(255), 150);
        ellipse(random(width), random(height), random(20, 80));
    }
}

Coding sebagai Medium Seni

Pemrograman bukan hanya tentang logika, tapi juga ekspresi kreatif. Pelajari bagaimana konsep TIK seperti:

  • Variabel dan Fungsi bisa membuat pola visual
  • Perulangan (loops) untuk menciptakan motif berulang
  • Kondisional (if-else) untuk seni interaktif
